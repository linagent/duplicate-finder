# DuplicateFinder Pro 🔍

智能重复文件查找工具 - 基于 Tauri + Rust + HTML/JS

## 功能特性

- ✅ 重复文件扫描（MD5哈希比对）
- ✅ 大文件查找
- ✅ 实时进度显示
- ✅ 3D 萨摩耶互动助手
- ✅ 3D 圆柱图表
- ✅ 安全删除（移到回收站）
- ✅ 图片预览

## 技术栈

- **前端**: HTML5 + CSS3 + TypeScript
- **后端**: Rust + Tauri
- **UI**: 自定义 Mac 风格设计

## 开发环境

### 前置要求

- [Node.js](https://nodejs.org/) (v18+)
- [Rust](https://rustup.rs/)
- [Tauri CLI](https://tauri.app/v1/guides/getting-started/prerequisites)
